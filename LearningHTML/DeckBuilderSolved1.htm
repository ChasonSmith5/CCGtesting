Card Building Solved is made by Democracy gaming a company dedicated to devloping games and tools to promote game devlopment with the intent of putting consumers first by making their voices heard.

Website / App Design

This application is designed to allow the user to choose any type of card game and test either existing cards in a game with different rule sets or create their own card and test how it may perform. On top of this this platform allows game devs to test balance futre cards to their games.
